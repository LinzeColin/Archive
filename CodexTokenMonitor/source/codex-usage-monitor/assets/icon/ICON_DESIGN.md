# Codex Token Monitor Icon

## Direction

The icon uses a business-professional monitoring metaphor:

- Dark graphite rounded-square base for a macOS-native, professional app feel.
- Cyan/blue circular allowance ring for quota-window monitoring.
- Layered token core for model token accounting.
- Small upward line chart for usage trend and dashboard analytics.

## Design Constraints

- No text or letters inside the icon.
- No OpenAI, Apple, or third-party logo reuse.
- High contrast at small menu, Dock, and Finder sizes.
- Suitable for dark and light macOS appearances.
